INSTALLATION STEPS (FRESH INSTALLATION)
================================================================================

1. PREPARE ENVIRONMENT
   - Install XAMPP (or Apache + MySQL + PHP)
   - Ensure Apache and MySQL are running
   - Verify PHP 7.4+ is installed

2. EXTRACT FILES
   - Extract sistem-tokohewan folder
   - Copy to c:\xampp\htdocs\ (on Windows)
   - Or /var/www/html/ (on Linux)
   - Or appropriate web root on your server

3. CREATE DATABASE
   - Open http://localhost/phpmyadmin/
   - Click "Import"
   - Select sql/petshop.sql
   - Click "Go"
   - Database "petshop" will be created with all tables

4. CONFIGURE CONNECTION
   - Edit config.php
   - Update database credentials if needed:
     * DB_HOST: 127.0.0.1 (or your MySQL server)
     * DB_NAME: petshop
     * DB_USER: root (or your MySQL user)
     * DB_PASS: "" (your MySQL password)

5. CREATE DEFAULT ADMIN
   - Open http://localhost/sistem-tokohewan/create_admin.php
   - Wait for "Default admin created" message
   - Do NOT run again (will show "Admin already exists")

6. TEST LOGIN
   - Open http://localhost/sistem-tokohewan/
   - Should redirect to login page
   - Login with: admin@petshop.local / admin123
   - Should redirect to admin dashboard

7. VERIFY INSTALLATION
   - Test all main menus
   - Verify database connectivity
   - Check that sample data appears
   - Test CRUD operations
