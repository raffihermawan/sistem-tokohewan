BACKUP & RECOVERY
================================================================================

Manual Backup:
  1. Open phpMyAdmin
  2. Select "petshop" database
  3. Click "Export"
  4. Save SQL file

Automatic Backup (Production):
  1. Setup cron job: mysqldump petshop > backup_$(date +%Y%m%d).sql
  2. Run daily at low-traffic time
  3. Store in secure location
  4. Verify restore weekly

Recovery:
  1. phpMyAdmin ‚Üí Import
  2. Select backup SQL file
  3. Click Go
  4. Database restored
