================================================================================
INSTALASI QUICK START
================================================================================

1. Extract folder ke c:\xampp\htdocs\sistem-tokohewan\

2. Import database:
   - Open phpMyAdmin
   - Create database "petshop" OR just import sql/petshop.sql
   - Tables akan terbuat otomatis

3. Config database (config.php):
   - DB_HOST: 127.0.0.1
   - DB_NAME: petshop
   - DB_USER: root
   - DB_PASS: "" (kosong untuk XAMPP default)

4. Generate admin:
   - Open http://localhost/sistem-tokohewan/create_admin.php
   - Wait for "Default admin created" message
   - JANGAN jalankan lagi

5. Login:
   - http://localhost/sistem-tokohewan/
   - admin@petshop.local / admin123
